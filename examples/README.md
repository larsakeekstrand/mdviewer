# MDViewer Examples

Open the `examples/` folder in MDViewer (**File ▸ Open Folder…**) to explore these fixtures in the sidebar. Each file is small enough to scan at a glance while showing enough syntax variety to exercise rendering and highlighting.

## markdown/

| File | Demonstrates |
|------|-------------|
| `kitchen-sink.md` | Headings h1–h4, bold/italic/inline-code, ordered and unordered lists, a table, a blockquote, a fenced code block, and links |
| `gfm.md` | GitHub-Flavored Markdown: task lists, aligned tables, strikethrough, and autolinks |
| `mermaid.md` | Two Mermaid diagrams — a flowchart and a sequence diagram |
| `math.md` | Inline math with `$…$` and a display block with `$$…$$` (KaTeX) |

## code/

| File | Demonstrates |
|------|-------------|
| `example.rs` | Rust: struct, impl, `fn main` with `println!` |
| `example.py` | Python: class, method with docstring, `if __name__ == "__main__"` |
| `example.js` | JavaScript: function, const, template literal, comment |
| `example.ts` | TypeScript: interface, typed arrow function |
| `data.json` | JSON: nested object, array, string, number, boolean, null |
| `example.css` | CSS: selectors, properties, media query, comment |
| `example.html` | HTML5: minimal document with `<script>` and `<style>` |
| `example.sh` | Bash: shebang, variable, for loop, comment |
| `example.yaml` | YAML: keys, nested map, list, comment |
| `example.go` | Go: package, import, struct, `func main` |
| `example.sql` | SQL: CREATE TABLE, INSERT, SELECT with JOIN, comment |
| `example.c` | C: #include, struct, function, `main()` |
| `notes.conf` | Unknown extension — line-numbered but uncolored fallback |
| `Makefile` | Extensionless — tests extensionless file handling |
